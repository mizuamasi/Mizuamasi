色と時間の変化

時間が経過するにつれて色が変化するシェーダーを作成しましょう。これにより、動的で生き生きとしたビジュアル表現が可能になります。

時間を使った色の変化

シェーダー内で時間の変数（`iTime`）を利用することで、色が時間とともに変化する効果を作り出します。これにより、静止画ではなく動きのある映像を生成できます。


// 時間によって色が変化するシェーダー
uniform float iTime;
uniform vec3 iResolution;

void mainImage(out vec4 fragColor, in vec2 fragCoord) {
    // ピクセルの座標を0から1の範囲に正規化
    vec2 uv = fragCoord / iResolution.xy;
    // 時間と座標に基づいて色を計算
    vec3 color = 0.5 + 0.5 * sin(iTime + uv.xyx + vec3(0, 2, 4));
    fragColor = vec4(color, 1.0);
}

void main() {
    mainImage(gl_FragColor, gl_FragCoord.xy);
}

このシェーダーでは、`sin`関数を使用して色が時間とともに変化します。`iTime`の値が増加するにつれて、色が滑らかに変わる様子を確認できます。

ピクセルの気持ちになって考える

想像してみてください。あなたは画面上の小さなピクセルです。毎フレーム、あなたは指揮官から新しい指示を受け取ります。しかし、あなたは自分の位置（座標）を知っています。指示には時間の変数も含まれており、あなたはそれを使って色を決定します。時間が経つにつれて、あなたの色も変わっていきます。このプロセスが連続して行われることで、動きのある美しいビジュアルが生まれます。
